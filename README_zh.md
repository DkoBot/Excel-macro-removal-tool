# 📋 程序结构说明文档

## 🛠️ 技术栈

| 项目 | 技术选型 |
|------|----------|
| **开发语言** | Python |
| **UI 框架** | PySide6 |
| **完成周期** | 1 个工作日 |

---

## ⚡ 多线程处理机制

### 核心技术
- **线程池技术**：采用高效线程池管理，彻底杜绝程序假死现象
- **主从分离架构**：主线程专注 UI 响应，后台线程全权负责文件处理任务
- **多语言支持**: 支持 中文/英文 切换国际化操作

### 性能表现
&gt; 💡 **典型应用场景**  
&gt; 即使同时处理 **500+** 个文档，程序依然保持流畅操作体验，响应无延迟。

## 📦 旧版本兼容性

| 特性 | 说明 |

| **格式支持** | 全面兼容 Office 97 至最新版本的宏文件解析 |
